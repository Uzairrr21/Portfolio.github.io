SignVerse — Bidirectional ASL Platform

Sep 2025 – May 2026 · Final Year Project · LGU

What it does

A browser-based ASL platform combining three modules: real-time sign-to-text recognition from webcam, text-to-sign 2D avatar animation, and an interactive 3-tier learning curriculum (easy → medium → hard). Fully deployed on Vercel + Hugging Face Spaces (Docker).

Architecture

  • Frontend: React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui. Firebase Auth + Firestore for user progress persistence.
  • Backend: FastAPI + PyTorch on Hugging Face Spaces (CPU Docker container).
  • Sign-to-Text: Browser captures 10fps frames → MediaPipe Holistic extracts 225-D landmarks (33 pose + 21+21 hand) → wrist-relative normalisation → 64-frame temporal resampling → Conv1D stem + 4-layer TransformerEncoder (d=256, 4 heads) → top-3 prediction from 250-class ASL vocabulary.
  • Text-to-Sign: Rule-based ASL gloss conversion → keyframe lookup from How2Sign + Google ASL database → 7 quadratic eased transition frames → Gaussian smoothing → 60fps 2D canvas avatar rendering.
  • Learning Module: Easy (A–Z CNN), Medium (full pipeline), Hard (Viterbi + Bigram LM for sentence recognition).
  • Extras: Google Translate multilingual output (6 languages), Google TTS audio playback.

Tea Leaf Disease Detection via Swin Transformer

2026 · Research Project · Paper Under Review

Results at a Glance

97.87%Test Accuracy
98.93%Val Accuracy
97 FPSTesla P100
TTA×7Augmentation

Methodology

  • Dataset: TeaLeafNet — 5,000 images, 4 disease classes.
  • Phase 1: Frozen Swin-T backbone, only classification head trained.
  • Phase 2: Stage-4 fine-tuning with MixUp regularisation (α=0.2), enabling deep feature adaptation.
  • Explainability: Grad-CAM with morphological isolation confirms disease-specific tissue localisation across all 4 classes — validated for clinical/agricultural use.
  • Inference: Test-Time Augmentation ×7 boosts accuracy to 97.87%. 97 FPS on Tesla P100 — viable for real-time field deployment.
  • Contribution: Establishes reproducible TeaLeafNet benchmark with full hyperparameter disclosure.

Urdu Fake News Detection — Hybrid Deep Learning

2025 · Co-authored: LGU & SMU Dallas, TX · Under Review

Results

83.33%Accuracy
0.8185Macro F1
+65ppMinority Recall
5,000Urdu Articles

Methodology

  • Tier 1 (Baseline): TF-IDF + Naive Bayes — establishes classical NLP baseline.
  • Tier 2 (Deep): DistilBERT-multilingual embeddings + classical classifiers (SVM, XGBoost).
  • Tier 3 (Ensemble): Weighted stacking ensemble achieving 83.33% accuracy and macro F1 0.8185, outperforming baseline by 13.33 pp.
  • Class Imbalance: SMOTE oversampling raised minority-class recall by 65 percentage points.
  • Dataset: 5,000 verified Urdu articles — low-resource language; first benchmark of this kind.

What it does
